# 👋 bonu — Portfolio

**Live sayt:** [bonush07.github.io](https://bonush07.github.io)

Bu mening shaxsiy portfolio saytim — full-stack dasturchi sifatida o'rganayotgan texnologiyalarim va loyihalarimni bir joyda jamlab ko'rsatish uchun yaratdim.

## 🧩 Sayt haqida

Sayt terminal/kod-muharriri uslubida ishlangan bitta sahifali (single-page) portfolio bo'lib, quyidagi bo'limlardan iborat:

- **Hero** — qisqa tanishtiruv, terminal animatsiyasi ko'rinishida
- **Tech Stack** — bilgan texnologiyalarim toifalarga bo'lib ko'rsatilgan (Frontend, Styling, Backend, Mobile)
- **Loyihalar** — GitHub'dagi eng so'nggi repositorylarimga havolalar
- **Bog'lanish** — GitHub profilimga link
